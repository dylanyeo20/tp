# CLIentTracker

![Ui](docs/images/Ui.png)

This is a **lightweight CLI CRM** that enables property agents to efficiently track their clients information such as contacts, property preferences/details, transaction information, manages events
within a streamlined interface.  ClientTracker makes it easy to organize and access client data, ensuring agents can focus on building relationships and closing deals.

---

### Quick start:
1. Download the latest version of the application [here](https://github.com/AY2526S2-CS2103T-T14-2/tp/releases)
2. Navigate to where you downloaded the file.
3. Run the application
   * Windows: Double-click on the JAR file
   * Linux/MacOS/Windows: run `java -jar app.jar`

For more detailed instructions, see the [User Guide](https://se-education.org/addressbook-level3/UserGuide.html#quick-start)

---

### Features
* List all contacts
* Add contacts
* Find contacts 
* Edit contacts
* Delete contacts
* Clear all entries
* Mark a specific contact
* Unmark a specific contact 
* Help command
